Description Min Max Speed Step Size Home Pos Battery Pos Servo Type ID Action

Descripion
Used in the servo input form to allow easy recognition
Min
Minimum Angular address for this type of servo - Typically zero for a 180 degree servo
Max
Maximum angular address for this type of servo - Typically something like 180 for a 180 Degreee Servo
Speed
Default traverse speed to be used for this type of servo - Lower value is faster, 5 fairly quick,25 fairly slow
Step Size
The number of degrees to step. Smaller numbers give smoother turning, larger numbers faster but with less resolution
Home Pos
Default position when the servo is out of use for this type of servo
Battery Pos
Default position when the servo is being controlled by a range finder but is out of scope for this type of servo
Servo Type ID
ID number of this servo type. Read only. Set once saved if empty
