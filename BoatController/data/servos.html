Descripion
Used for identification in RC Channels setup
Controller
Which controller the servo is attached to
Port Number
Which controller port number the servo is attached to
Servo Type
Base class of servo from the Servo Types screen
Controlling Servo
A servo which will move this servo as it moves - Typically the ID of a Range Finder type servo
Reverse
Reverse the direction of the servo
Absolute
Does the servo use absolute or relative positioning - Absolute the angle is determined by the RC channel angle - Relative is incremental
Tracker
When set to a non-home position does this servo track the compass
Center Angle
What angle to the centreline of the hull does this servo face when centered (0-359 180 means facing directoy to rear)
Speed
Custom traverse speed for this servo - Lower value is faster, 5 fairly quick,25 fairly slow
Step Size
Custom Step Size for this servo - Lower value is slower but with better resolution
Servo ID
System Identifier for servo (Read only,set on save if not present)
